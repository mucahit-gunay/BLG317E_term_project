Public Transportation GTFS Data of Istanbul

Team: TESSERACT | BLG317E Database Systems

About the Project

This project aims to design a relational database based on the GTFS (General Transit Feed Specification) format for Istanbul's public transportation network. It provides structured information about agencies, routes, stops, trips, and schedules.

Dataset

Istanbul Public Transport GTFS Data
